# 🌸 Iris Softmax Classifier

A production-ready **Softmax Regression (Multinomial Logistic Regression)** classifier trained on the classic **Iris dataset**, served via a **FastAPI** microservice and deployed on **Google Cloud Run**.  
The project includes a full **CI/CD pipeline using GitHub Actions** for automated testing, containerization, and deployment.

## 🚀 Overview

This repository demonstrates an end-to-end **MLOps workflow**:

- **Data Processing:** Loads and prepares the Iris dataset for training.
- **Model Training:** Trains a softmax classifier using scikit-learn.
- **Artifact Management:** Saves the trained model as `artifacts/model.pkl`.
- **Serving:** Exposes a `/predict` REST endpoint via FastAPI.
- **Deployment:** Uses Docker and Google Cloud Run.
- **CI/CD:** Automates linting, testing, image building, and deployment with GitHub Actions.

## 🐳 Containerization & Deployment

**Build the image locally:**

```bash
docker build -t iris-softmax:latest .
```

**Run locally:**

```bash
docker run -p 8080:8080 iris-softmax:latest
```

**Deploy to Google Cloud Run:**

```bash
export PROJECT_ID="your-project-id"
export REGION="us-central1"
export REPO="iris-repo"
export IMAGE="us-central1-docker.pkg.dev/${PROJECT_ID}/${REPO}/iris-softmax:latest"
export SERVICE_NAME="iris-softmax-service"

# Create repo (if not exists)
gcloud artifacts repositories create ${REPO} --repository-format=docker --location=${REGION} || true

# Build & push image
gcloud builds submit --tag ${IMAGE}

# Deploy to Cloud Run
gcloud run deploy ${SERVICE_NAME} \
  --image=${IMAGE} \
  --region=${REGION} \
  --platform=managed \
  --allow-unauthenticated
```

## 🌐 Example Request

After deployment, get your Cloud Run URL and test:

```bash
SERVICE_URL=$(gcloud run services describe iris-softmax-service \
  --region us-central1 --platform managed \
  --format='value(status.url)')

curl -X POST "${SERVICE_URL}/predict" \
  -H "Content-Type: application/json" \
  -d '{
    "sepal_length": 5.1,
    "sepal_width": 3.5,
    "petal_length": 1.4,
    "petal_width": 0.2
  }'
```

**Expected response:**

```json
{ "species": "setosa" }
```
